# 🚗 Car Price Prediction System

A Machine Learning application that predicts the selling price of a used car based on features such as manufacturing year, fuel type, transmission, mileage, engine capacity, owner details, and more.

The project includes data preprocessing, exploratory data analysis (EDA), model training, evaluation, hyperparameter tuning, and an interactive Streamlit web application.

## 📦 Dataset

The dataset is provided in **RAR (.rar)** format.

### Steps to use the dataset

1. Download or clone this repository.
2. Extract the **RAR** file using software such as:
   - WinRAR
   - 7-Zip
3. After extraction, copy the **car_data.csv** file into the **data/** folder.

The folder should look like:

```
data/
└── car_data.csv
```

> **Note:** The project will not work directly with the `.rar` file. You must extract it first and place the CSV file inside the `data` folder.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Car-Price-Prediction.git
```

Go to the project directory

```bash
cd Car-Price-Prediction
```

Install the required packages

```bash
pip install -r requirements.txt
```

Train the Machine Learning model

```bash
python src/train.py
```

Run the Streamlit application

```bash
streamlit run app.py
```
